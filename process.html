Process of the Project

Project Proposal

The process of creating the game consits of:
  1. Learning the basic syntax of Python
  2. Installment of PyCharm CE
  3. Reading the textbook of Pyarcade
  4. Reading the libraries of Pyarcade
  5. Watching Pyarcade tutorials on Youtube
  6. Reading different Pyarcade tutorials online
  7. Trying to code with different tutorials
  8. Choosing the best choice of Pyarcade game tutorial: Simple Platformer
  9. Following the game tutorial on the Pyarcade Website
  10. Coding different sprites, actions, sound effects, screen movements, etc.
  11. Debugging and Modifications of the game
  12. Asking people around me to play and give feedback
